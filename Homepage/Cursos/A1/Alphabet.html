El abecedario

El abecedario es una de las partes principales por las cuales se empieza para aprender un lenguaje nuevo, inclusive el lenguaje que adquirimos de niños. Es importante resaltar que el abecedario es distinto en diferentes magnitudes dependiendo del idioma que se desea aprender, en este caso el inglés, tiene mucha similitudes con respecto a la simbología que se presentan en el abecedario español solo que este excluye la letra Ñ además de que el fonema cambia, es decir, no suenan igual al abecedario español.

Importancia del alfabeto

El alfabeto nos permite reconocer y asociar símbolos (en este caso letras) con sonidos, regalando un primer acercamiento al aprendizaje a una lengua, funciona para muchas cosas básicas como deletrear nombres, direcciones, ayudar a la pronunciación, pensar en el idioma y familiarizarse desde temprano con el fonema que se maneja.
